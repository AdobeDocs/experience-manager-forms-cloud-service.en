# Using Designer {#install-designer}

Designer is a point-and-click graphical form design tool that simplifies the creation of forms.  You can design a form, define its logic, and modify it to match paper counterparts or to meet strict legislative requirements. You can also use the Designer to create a Document of Record template for an adaptive form. The following documents provide instructions to use designer to create XFA-based PDF Forms templates:
